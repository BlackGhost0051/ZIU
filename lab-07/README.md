# Laboratorium 7 - Formularze i walidacja danych

## Szybki start

### Instalacja i uruchomienie

```bash
npm install
npm run dev
```

### Dostęp do aplikacji

- **Aplikacja Todo (Lab 6)**: http://localhost:3000/
- **Formularz rejestracji (Lab 7)**: http://localhost:3000/register

## Opis projektu

Wieloetapowy formularz rejestracyjny z walidacją React Hook Form + Zod i pełną dostępnością WCAG 2.1 AA.
